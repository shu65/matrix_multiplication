matrix_multiplication
=====================

Sample of matrix multiplication on CPU and Xeon Phi

QuickStart
----------
    cd ./src
    make Mic=Y
    
CPU mode 
-----------
    ./matrix_multiplication 1024 0
    
Xeon Phi mode
-----------
    ./matrix_multiplication 1024 1
